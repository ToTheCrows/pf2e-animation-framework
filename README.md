# pf2e-animation-framework
This FoundryVTT Modul took heavy inspiration from [PF2e Graphics](https://github.com/MrVauxs/pf2e-graphics). But i wanted something less complicated for me and all of you to use and integrate into future Foundry versions. Atm one big clean script with an Animation Map and a robust logic to find the right animation, when it´s needed.
